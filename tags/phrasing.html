Phrasing Tags Test Page

About

Phrasing tags are used to mark up the content of a document.

Tag <a>: Add link to a page

Click here to start your journey online.

Tag <abbr>: Abbr for abbreviation

Move your cursor over the letters to see what the abbreviation means: TGIF.

Tag <area>: Use area together with <map>

Click on the chicken will direct you to a info page.

The image was created by m5bq.

Tag <b>: Bold text

It is a true pleasure to do homework on the Chinese New Year eve.

Tag <bdo>: Bidirectional text with overridden direction

I am making a mirror text here here

Tag <bdi>: Bidirectional isolation text

Like the <bdo> tag but detects the direction automatically (depending on the language): صباح الخير.

Tag <br>: Line break element. no content and no end tag

Today is a good day
I just used a line-breaker! Yay!

Tag <cite>: Used for title of a work, author names, etc

Here we demonstrated together with tag <blockquote>.

Tradition becomes our security, and when the mind is secure it is in decay.
Jiddu Krishnamurti

Tag <code>

We always need a main() function.

Tag <data>: To make machine-readable data to be displayed user-friendly

Checkout book list:

  • The Indian Music
  • How to cook in 3 minutes
  • Introduction to HTML

Tag <datalist>: Usually used as a list for user to choose from

What would you like to eat on the New Year's eve:

Tag <del> and <ins>: Remove text and insert text

Note that the default render for <ins>is to underline text. However, semantically, it denotes an element which has been inserted.

  1. buy rice cake
  2. go to BOA
  3. finish homework

Tag <s>: Delete irrelevant text

The result is the same as tag <del>, but

  • <del> represents a removal from a document
  • <s> represents contents that no longer accurate or relevant

Lunch buffet for only $10 $12 per person.

Tag <u>: Underline

This tag is used for presentation purpose.

Tag <dfn>: First time defining an word

Red envelopes are passed out during the Chinese New Year's celebrations. A Red envelopes is an red packet with money, which is used to suppress the evil spirit, literally.

Tag <em>: Used for emphasizing words

If you do not do this homework, you will fail the class.

Tag <i>: Used for italic names, technical terms, etc

The stranger said her name was Sophie.

Tag <kbd> and <samp>: Represents input text that a user should type

Useful in a instruction file. display text that the user should enter into their keyboard. Here we demonstrated with <samp> tag which usually used to represent sample output of a program.

How to search online

  1. open your browser
  2. type Google.com into your browser address bar.
  3. click Enter

Tag <mark>: Highlighting text to draw attention

I ate 100000000 burgers today.

Tag <meter>: A gauge that represents a percentage or a range

Homework 3 finished:

Tag <progress>: A bar that represents the progress of a task

Scanning now. progress: 20 %

Tag <output>: Represents the result of a calculation

x =

Tag <q>: Quote

Jiddu Krishnamurti said It is no measure of health to be well adjusted to a profoundly sick society.

Tag <ruby>: Ruby annotation. common use for eastern language

Often used with tag <rt>, which can be used for explanation or pronunciation: 日に本ほん語ご

Tag <small>: Small font used for legal restriction, copyright and so on

© Copyright 2017, FLJ Corporation.

Tag <strong>: To convey urgency and importance

Warning: Late submission will not be accepted!

Tag <time>

The homework is due on 2017-01-28 20:59.

Tag <var>: Variable, could be used in math equation

Let's assume x stands for the total income.

Tag <sub>: Subscript text

This tag should only be used when the absence of the element will change the meaning of the content. It should not be used only for presentation purposes.

Carbon Dioxide is often written as CO2.

Tag <sup>: Superscript text

One of the basic unit in measuring volume is m2.

Tag <span>: Meaning nothing on its own

Span is often used with other global attributes. For example, using attribute class to classify if as a keyword: if.
